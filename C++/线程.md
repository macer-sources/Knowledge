## c++ 标准库 
```c++
#include <thread>

// 定义thread 函数 
void  threadFunction(int i) {
  // 执行子线程的任务
}

std:: thread t;
// 
std::thread t(threadFunction, 10);
 t.join();// 执行线程， 会阻塞当前发起的线程 ，系统不会回收(t.joinable() 处于此状态，系统会报异常)
t.detach();// 将子线程和发起线程分离， 系统会进行线程回收
```
